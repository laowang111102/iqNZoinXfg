# 教务管理系统技术文档 python880

## 项目概述

教务管理系统是基于 Python 语言和 Flask 框架开发的，旨在提供高效、便捷的教务管理解决方案。

## 技术栈

- **后端**: Python 3.x
- **Web 框架**: Flask
- **数据库**: MySQL
- **运行环境**: PHPStudy（集成环境）

## 功能模块

- **班级管理**
  - 班级信息维护
  - 学生信息管理
- **课表管理**
  - 课程信息发布
  - 课表查询

## 系统角色

- 管理员：负责教务信息的维护和管理
- 教师：可查询课表信息，进行课程管理
- 学生：查询个人课表信息

## 运行环境

- **操作系统**: Windows/Linux均可
- **环境要求**: PHPStudy 集成环境，包含 Python、MySQL

## 部署步骤

1. 安装并配置 PHPStudy 环境
2. 克隆或下载项目代码
3. 导入 MySQL 数据库结构
4. 修改配置文件，连接本地数据库
5. 运行 Flask 应用

## 目录结构

```
project/
│
├── app/                 # Flask 应用目录
│   ├── static/          # 静态资源目录
│   ├── templates/       # 模板文件目录
│   └── views/           # 视图层目录
│
├── config.py            # 配置文件
├── models.py            # 数据模型文件
└── run.py               # 应用启动文件
```

## 核心亮点

- **轻量级**: 基于 Flask 框架，系统轻量，易于维护和扩展。
- **灵活性**: 采用 Python 开发，方便定制和二次开发。
- **跨平台**: 适用于 Windows/Linux 等多种操作系统。
- **易用性**: 提供简洁友好的用户界面，降低使用门槛。
